🧩 The Problem: Given n, find three non-negative integers a, b, c such that:
a + b + c = n
That's literally it. Output any valid triple! 😄

💡 The "Theorem" — 0 + 0 + n = n:
The "Hexadecimal's Theorem" being referenced here is simply:
Every non-negative integer n can be written as 0 + 0 + n

The checker confirms: 0 + 0 + n = n ✅ for any n!
The problem is completely trivial — but it's a beautiful reminder that sometimes the simplest mathematical truth (the existence of zero!) is all you need! 🧮
