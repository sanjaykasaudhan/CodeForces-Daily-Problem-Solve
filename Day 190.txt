✅ Accepted on Codeforces! Problem A — Bingo Candies | Round 1086 (Div. 2)

Bingo cards, frequency counting, and a beautiful upper bound check — this problem is a perfect example of how data structure choice simplifies everything! 🎰🍬

🔗Problem Link: https://lnkd.in/gJ3aGG9M

🧩 The Problem: You have n bingo cards, each with n numbers. A number can appear on multiple cards. Determine if it's possible to call out numbers such that every card gets bingo (i.e., every card has at least one called number)!
The key insight: you need to call at least one number from each card. The maximum frequency of any single number tells you the "best" number to call — it covers the most cards at once!
YES if and only if: the maximum frequency ≤ n*(n-1) (you can spread calls optimally) — but wait, the actual condition from the code is:
mx <= n * (n - 1)
where mx = max(freq.values()) — the maximum occurrence count across all numbers! 🎯

💡 My Approach — Counter + Frequency Max:
Step 1: Build a Counter by reading all n rows (each with n numbers):
freq = Counter()
for _ in range(n):
    freq.update(map(int, input().split()))
Step 2: Find the maximum frequency:
mx = max(freq.values())
Step 3: Check feasibility:
print("YES" if mx <= n * (n - 1) else "NO")

Examples:
n=3, cards=[[1,2,3],[3,1,4],[4,1,2]] → freq: 1→3,2→2,3→2,4→2 → mx=3, n*(n-1)=6 → 3≤6 → YES ✅
n=3, cards=[[1,1,1],[2,2,4],[1,4,3]] → freq: 1→4,2→2,3→1,4→2 → mx=4, 6 → 4≤6 → YES ✅
n=3, cards=[[1,1,1],[1,1,1],[1,1,2]] → freq: 1→8,2→1 → mx=8, 6 → 8>6 → NO ✅

📌 Key Takeaways:
🎰 Counter.update() for multi-row frequency counting — calling update() multiple times accumulates counts across all rows. One Counter for the entire grid! 🎯
🎰 max(freq.values()) — finds the most common element across all cards in one call! ✅
🎰 n*(n-1) is the critical threshold — an n×n bingo grid has n² total cells across n cards. If one number appears in more than n*(n-1) positions, it's dominating too many cards! 🧮
🎰 freq.update(map(int, input().split())) — combining update() with map() in one line is the cleanest way to add an entire row to a Counter! 🐍
