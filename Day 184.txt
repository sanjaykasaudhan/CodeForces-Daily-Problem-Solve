🧩 The Problem: Given n, output the sequence n, n-1, n-2, ..., 2, 1 (counting down from n to 1)!
That's the entire problem. And the code:
print(*range(n, 0, -1))
range(n, 0, -1) generates n, n-1, ..., 2, 1 and * unpacks it for space-separated output! 🎯

💡 The Python One-Liner Magic:
range(n, 0, -1) — three arguments: start=n, stop=0 (exclusive), step=-1 → counts down from n to 1!
print(*range(n, 0, -1)) — the * unpacking spreads all values as separate arguments to print, which automatically adds spaces between them! ✅

Examples:
n=2 → 2 1 ✅
n=3 → 3 2 1 ✅
n=4 → 4 3 2 1 ✅
n=5 → 5 4 3 2 1 ✅

📌 Key Takeaways:
📝 range(start, stop, step) mastery — the three-argument form of range() is one of Python's most powerful tools. range(n, 0, -1) for countdown, range(0, n, 2) for evens, etc. Master all forms! 🎯
📝 print(*iterable) — unpacking any iterable into print() gives space-separated output. This replaces ' '.join(map(str, ...)) with a more concise form! ✅
📝 Recognise trivial sequence outputs — "print n down to 1" = range(n, 0, -1). Build a mental catalogue of common sequences and their Python generators! 🐍
