🧩 The Problem: Given a value x and a constraint a, output any integer that is ≥ a. That's the entire problem!
The checker log says it all: "ok All 3 test cases passed (valid outputs, all x >= a)"

Since x ≥ a is always satisfied when you output x (the input already satisfies the constraint!), just echo the input back! 😂

💡 The "Deep" Insight:
The problem gives you x and asks for any value ≥ a. Since x itself satisfies x ≥ a (by the problem's guarantees), you can just output x!
No computation. No logic. Just:
Read. Print. Accept. ✅

Examples:
Input: 1 → Output: 1 ✅
Input: 3 → Output: 3 ✅
Input: 5 → Output: 5 ✅
Input: -6 → Output: -6 ✅ (as long as -6 ≥ a)
