🧠 Problem Summary
You’re given:
An array of size n
A target sum s
A number x
You can increase any array element by x any number of times.
 The task is to determine whether it’s possible to make the array sum exactly equal to s.

💡 Key Observation
Let:
current_sum = sum(array)
Then:
If current_sum > s → ❌ impossible
Else, check if the difference (s - current_sum) is divisible by x
That’s it. No simulation needed ✔️

If s ≥ current_sum and (s - current_sum) % x == 0 → YES
Else → NO

Simple math + clean logic ✨
