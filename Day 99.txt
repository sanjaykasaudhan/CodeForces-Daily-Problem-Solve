🧠 Problem Idea:
Given a binary string, we repeatedly remove pairs of 01 or 10.
Each removal eliminates one 0 and one 1 from the string.
So the remaining length after all possible removals is:
n − 2 × min(count(0), count(1))

💡 Approach:
Count the number of zeros and ones in the string.
Determine how many pairs can be removed.
Compute the final remaining length using the formula.

🔑 Key Learning:
Many string problems become simpler when converted into counting problems instead of simulations.
