🧠 Key Idea:
The game removes adjacent 01 or 10. Each move removes one 0 and one 1.
So the total number of possible moves = min(count(0), count(1)).
If the number of moves is odd → Alice wins (DA)
If the number of moves is even → Bob wins (NET)

💡 What this problem teaches:
Convert a game simulation into a simple counting problem
Look for patterns instead of brute force
